# 🚀 Tlamatiliz Search

> **Tu centro de conocimiento.**

## 📖 Descripción

**Tlamatiliz Search** es una plataforma educativa de vanguardia desarrollada para el **Learn-A-Thon 2025**. Su objetivo es democratizar el acceso al aprendizaje personalizado mediante el uso de modelos de lenguaje avanzados. La aplicación permite a los usuarios generar cursos sobre cualquier tema al instante, aprender lógica de programación de manera interactiva y gestionar su progreso en la nube.

## ✨ Características Principales

- **🔍 Buscador Académico**: Motor de búsqueda inteligente que filtra resultados comerciales para ofrecer fuentes académicas y verificadas para tus investigaciones y tareas.
- **☁️ Sincronización en la Nube**: Tu progreso, cursos creados y configuraciones se guardan automáticamente con **Firebase**.
- **🔐 Autenticación Robusta**: Sistema de login y registro seguro.
- **👥 Comunidad**: Espacio para conectar con otros estudiantes (Discord/WhatsApp).
- **🎨 Diseño Moderno**: Interfaz oscura con estilo *Glassmorphism*, animaciones fluidas y experiencia de usuario pulida.

## 🛠️ Tecnologías Utilizadas

Este proyecto ha sido construido utilizando tecnologías web modernas y servicios en la nube:

- **Frontend**: HTML5, JavaScript (ES6 Modules), CSS3.
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/) + CSS personalizado.
- **Backend as a Service (BaaS)**: [Google Firebase](https://firebase.google.com/) (Auth & Firestore).
- **Iconos y Tipografía**: FontAwesome, Google Fonts (Inter, Poppins).
  
## 🚀 Instalación y Uso

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/nexus-search.git
   ```
2. **Configuración**:
   - El proyecto utiliza Firebase. Asegúrate de configurar tus propias API Keys si planeas desplegar tu propia versión.
3. **Ejecutar**:
   - Al ser una aplicación web estática con módulos JS, se recomienda usar un servidor local (como Live Server en VS Code) para evitar errores de CORS.
   - Abre `index.html` en tu navegador.
